# SPA - Boilerplate
###  Single Page Application - boilerplate,
Project structure for starting a SPA project.

## INSTALL:
    git clone https://github.com/dragosh/spa-boilerplate.git project-name.dev
    cd project-name.dev
    npm install && bower install
